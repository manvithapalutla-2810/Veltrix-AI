# 🚀 Veltrix AI

> AI-Powered Recruitment Intelligence Platform

🌐 **Live Demo:** https://veltrix-ai.streamlit.app/

---

## 📌 Overview

Veltrix AI is an AI-powered recruitment platform that automates resume screening and candidate evaluation. It helps recruiters analyze resumes, calculate ATS scores, rank candidates, identify skill gaps, generate AI-powered feedback, and send interview emails—all through an intuitive web application.

---

## ✨ Features

- 🤖 AI Resume Analysis using Google Gemini
- 📄 Resume Parsing (PDF)
- ⭐ ATS Score Calculation
- 🎯 Resume Matching using TF-IDF & Cosine Similarity
- 📊 Recruitment Dashboard
- 📈 Interactive Analytics
- 🏆 Candidate Ranking
- 📧 Automated Email Notifications
- 📄 PDF Report Generation
- 📥 CSV Report Export

## 🚀 Installation

Clone the repository:

```bash
git clone https://github.com/manvithapalutla-2810/Veltrix-AI.git
```

Move into the project:

```bash
cd Veltrix-AI
```

Install dependencies:

```bash
pip install -r requirements.txt
```

Run the application:

```bash
streamlit run app.py
```
